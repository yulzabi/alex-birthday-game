## 🎮 על המשחק

משחק דפדפן בו הדמות זזה על המסך ואוספת מתנות, כוכבים, עוגות ופריטים שמחים נוספים.
מותאם לגיל 5 — שליטה פשוטה, צבעים עליזים, והודעות עידוד!

### תכונות:
- 🎁 **3 שלבים** של איסוף פריטים עם קושי מתקדם
- ⭐ **10 סוגי פריטים** — מתנות, כוכבים, עוגות, פרפרים, כתרים ועוד
- 🎵 **צלילים שמחים** כשאוספים פריט או עוברים שלב
- 💬 **הודעות עידוד** בעברית — "כל הכבוד!", "את כוכבת!", "מלכה!"
- 🌈 **רקע משתנה** בכל שלב (כחול → ורוד → זהוב)
- 📱 **תמיכה במובייל** — כפתורי מגע למסכי מגע
- 🖼️ **תמונה מותאמת** — אפשר להחליף את האימוג'י בתמונה אישית

## 🚀 איך לשחק

### אפשרות 1: פתיחה ישירה
פשוט פתחו את `index.html` בדפדפן!

### אפשרות 2: שרת מקומי
```bash
# אם מותקן Python
python -m http.server 8000

# או עם Node.js
npx serve
```

### שליטה:
| מכשיר | שליטה |
|--------|--------|
| 💻 מחשב | חצים או מקשי WASD |
| 📱 מובייל | כפתורי כיוון על המסך |

## 🖼️ הוספת תמונה אישית

1. שמרו תמונה בשם `alex.png` בתיקיית `images/`
2. רצוי תמונה מרובעת (למשל 200x200 פיקסלים)
3. רקע שקוף (PNG) ייראה הכי טוב, אבל כל פורמט יעבוד
4. התמונה תחליף אוטומטית את האימוג'י 👧

## 📁 מבנה הפרויקט

```
alex-birthday-game/
├── index.html          # דף HTML ראשי
├── css/
│   └── style.css       # עיצוב ואנימציות
├── js/
│   ├── audio.js        # מנוע סאונד (Web Audio API)
│   ├── config.js       # הגדרות משחק (פריטים, צבעים, הודעות)
│   ├── controls.js     # ניהול קלט (מקלדת + מגע)
│   ├── game.js         # לוגיקת משחק (שחקן, פריטים, התנגשויות)
│   ├── intro.js        # קישוטי מסך פתיחה (קונפטי, בלונים)
│   ├── main.js         # קובץ ראשי — מחבר הכל
│   └── renderer.js     # מנוע ציור (Canvas API)
├── images/
│   └── alex.png        # (אופציונלי) תמונת השחקנית
└── README.md
```

## 🛠️ טכנולוגיות

- **HTML5 Canvas** — לציור גרפיקה
- **Web Audio API** — ליצירת צלילים בזמן אמת
- **CSS Animations** — לאנימציות מסך הפתיחה והניצחון
- **Vanilla JavaScript** — ללא ספריות חיצוניות!

## 🎨 התאמה אישית

ניתן לשנות בקלות:

### `js/config.js`:
- `ITEMS_PER_LEVEL` — כמות פריטים בכל שלב
- `TOTAL_LEVELS` — כמות שלבים
- `ITEM_TYPES` — סוגי פריטים ונקודות
- `ENCOURAGEMENTS` — הודעות עידוד
- `LEVEL_COLORS` — צבעי רקע לכל שלב

### `index.html`:
- שם הילדה/ילד
- גיל
- טקסטים
